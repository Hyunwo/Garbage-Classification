## Overview
### Services
- 검색을 통해 분리수거 방법 찾기 (Search for Recycling)
- 이미지 삽입을 통해 분리수거 방법 찾기 (Insert image for Recycling)
- 자유게시판을 통해 사람들과 의견을 나눠 분리수거 방법 찾기 (Sharing opnion for Recycling)
